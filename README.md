# Trust-Centric Threat Modeling (TCTM)

A framework for evaluating security, governance and decision integrity in autonomous AI systems.

## Research Areas

- AI Security
- AI Governance
- Trustworthy AI
- Agentic AI
- Cybersecurity Strategy

## Framework Components

- Trust Domains
- Threat Taxonomy
- Trust Risk Matrix
- Trust Maturity Model
