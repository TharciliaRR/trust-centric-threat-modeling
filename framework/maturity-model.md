# Trust Maturity Model

The TCTM Trust Maturity Model evaluates how prepared an organization is to govern, monitor, and preserve trust in autonomous AI systems.

| Level | Description | Capability |
|------|-------------|------------|
| Level 1 | Experimental | Trust is assumed, not measured |
| Level 2 | Controlled | Basic controls exist |
| Level 3 | Governed | Governance processes are formalized |
| Level 4 | Resilient | Trust is continuously monitored |
| Level 5 | Trust-Centric Enterprise | Trust is a strategic capability |
