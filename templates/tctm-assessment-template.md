# 7. Trust Risk Assessment

## Risk Scoring Formula

```text
Risk Score = Impact × Likelihood × Trust Impact
```

---

## Impact Scale

| Score | Description |
| ----- | ----------- |
| 1     | Negligible  |
| 2     | Minor       |
| 3     | Moderate    |
| 4     | Significant |
| 5     | Severe      |

---

## Likelihood Scale

| Score | Description    |
| ----- | -------------- |
| 1     | Rare           |
| 2     | Unlikely       |
| 3     | Possible       |
| 4     | Likely         |
| 5     | Almost Certain |

---

## Trust Impact Scale

| Score | Description                  |
| ----- | ---------------------------- |
| 1     | Minimal trust degradation    |
| 2     | Limited confidence reduction |
| 3     | Noticeable trust impact      |
| 4     | Significant trust erosion    |
| 5     | Critical loss of confidence  |
