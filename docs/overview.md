# Framework Overview

The Trust-Centric Threat Modeling Framework (TCTM) was developed to address risks introduced by autonomous AI systems.

Traditional threat modeling frameworks focus primarily on protecting infrastructure, applications, and information.

TCTM extends this perspective by introducing trust as a primary assessment objective.

The framework evaluates:

* Identity Trust
* Context Trust
* Decision Trust
* Governance Trust
* Operational Trust

The goal is to help organizations preserve confidence in autonomous decision-making environments.
