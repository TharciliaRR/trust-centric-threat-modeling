# Threat Taxonomy

The TCTM Threat Taxonomy classifies threats according to their impact on trust.

---

# Threat Categories

| Category | Description                |
| -------- | -------------------------- |
| TMT      | Trust Manipulation Threats |
| DIT      | Decision Integrity Threats |
| GFT      | Governance Failure Threats |
| OTT      | Operational Trust Threats  |

---

# Trust Manipulation Threats (TMT)

Threats designed to influence confidence and trust.

Examples:

* Prompt Injection
* Context Poisoning
* Confidence Manipulation

---

# Decision Integrity Threats (DIT)

Threats affecting decision quality.

Examples:

* Hallucinated Actions
* Decision Drift
* Misalignment

---

# Governance Failure Threats (GFT)

Threats targeting accountability and oversight.

Examples:

* Missing Accountability
* Governance Bypass
* Audit Gaps

---

# Operational Trust Threats (OTT)

Threats affecting runtime trustworthiness.

Examples:

* Agent Hijacking
* Runtime Manipulation
* Resilience Failures

---

# Purpose

The taxonomy supports systematic identification of threats impacting autonomous AI trustworthiness.
