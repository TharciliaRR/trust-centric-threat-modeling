# Research Papers

This directory contains academic-style papers derived from the Trust-Centric Threat Modeling Framework (TCTM).

The objective is to translate TCTM concepts into publication-ready research artifacts suitable for conferences, workshops, journals, and academic collaboration.

## Current Papers

### TCTM Foundation Paper

Introduces the Trust-Centric Threat Modeling Framework.

### Decision Security

Explores decision integrity as a cybersecurity objective.

### Trust Observability

Investigates monitoring trust in autonomous AI systems.

### Agent Governance

Examines governance structures for autonomous agents.
